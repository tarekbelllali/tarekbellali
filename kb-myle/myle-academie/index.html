FAX-WEB

1 – Can all types of medical professionals use the fax-web or is it only for doctors?

All medical professionals who have the permission can use the fax-web, can re-sent fax from portfolio
tab if there’s an issue.

2 – How many pages can one send in one fax?

between 20-30 pages for optimal sending fax, if we send more the temp memory fax will be more and
more lazy

3 – How can I change my PIN?

Click on (Nip) setup the 4 digit code to activate NIP and protect sending fax. PS: Needs Double confirmation

4- How can I add my signature?

Yes, we are able to add our signature, we click on signature and import png file from our device

5- How, and where can I track my statuses?

On Portfolio tab, we can check if the fax is sent or not, also there’s a red color showing if (fax issue)

6- Why can’t I see the fax option, after adding the PIN number and my signature?

To see fax options, we have to make sure that fax number is checked on configuration

7- Can they fax out to people who aren’t medical professionals? (ex: lawyers, housing .. )

Yes, people who aren’t medical professional can receive faxes from fax web

8- Can we receive faxes in MYLE, or only send out?

We are only able to send fax from Myle

9- What are the documents that can be faxed ?

Prescriptions, forms.
